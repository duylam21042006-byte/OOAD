# 📅 Calendar Appointment Manager

Một ứng dụng quản lý lịch hẹn cá nhân đơn giản và hiệu quả, được xây dựng bằng Java Swing. Ứng dụng giúp bạn dễ dàng theo dõi và quản lý các cuộc hẹn, cuộc họp, và sự kiện quan trọng trong cuộc sống hàng ngày.

## ✨ Tính Năng

- **Xem lịch theo tháng**: Hiển thị lịch với các ngày trong tháng, dễ dàng điều hướng.
- **Thêm lịch hẹn**: Tạo mới các cuộc hẹn với thông tin chi tiết như tiêu đề, địa điểm, thời gian, và ghi chú.
- **Quản lý cuộc hẹn**: Xem danh sách các cuộc hẹn trong ngày đã chọn.
- **Nhắc nhở thông minh**: Hỗ trợ đặt nhắc nhở cho các cuộc hẹn quan trọng.
- **Họp nhóm**: Quản lý các cuộc họp nhóm với danh sách người tham gia.
- **Xử lý xung đột**: Tự động phát hiện và xử lý các cuộc hẹn trùng thời gian.
- **Lưu trữ cục bộ**: Dữ liệu được lưu trữ an toàn trên máy tính của bạn.

## 🚀 Cách Chạy

### Yêu Cầu
- Java JDK 17 hoặc cao hơn
- IDE như IntelliJ IDEA, Eclipse, hoặc VS Code

### Hướng Dẫn
1. Clone repository này về máy:
   ```bash
   git clone https://github.com/duylam21042006-byte/OOAD.git
   cd OOAD
   ```

2. Mở project trong IDE và chạy class `CalendarApp.java` trong thư mục `src`.

3. Ứng dụng sẽ khởi động và hiển thị giao diện chính.

## 🏗️ Kiến Trúc

Ứng dụng được thiết kế theo mô hình MVC (Model-View-Controller):
- **Model**: `CalendarModel` quản lý dữ liệu, `Appointment` đại diện cho cuộc hẹn.
- **View**: `CalendarApp` và `CalendarPanel` xử lý giao diện người dùng.
- **Controller**: Logic xử lý trong các lớp chính.
